Pra você só colar o link e tudo acontecer sozinho — sem terminal, sem git — o dashboard precisa de permissão pra salvar no seu próprio repositório do GitHub. Isso é feito com um token de acesso pessoal, gerado uma única vez e guardado só neste navegador (nunca é enviado a mais ninguém além do GitHub).

  1. Abra github.com/settings/personal-access-tokens/new ↗
  2. Em "Repository access", escolha Only select repositories → selecione este repositório ()
  3. Em "Permissions → Repository permissions", ache Contents e mude para Read and write
  4. Clique em "Generate token" e copie o valor (começa com github_pat_…)

Adicionado! O GitHub Actions já foi disparado automaticamente — em cerca de 1 minuto a contagem de anúncios aparece aqui (atualize a página).
